## Healthcare Contributions

- Listened to frontline hospital staff to understand clinical and OPD workflows  
- Traced healthcare databases (50+ tables, 1M+ records) using SQL with limited documentation  
- Supported 43-file reporting based on nursing requirements, increasing reporting frequency to 3x per week  
- Verified patient eligibility via NHSO before OPD workflow submission  
- Used LLMs to infer HOSxP schema (e.g., DTX) and locate data within 24 hours  

## Additional Impact

- Supported CID/HN registration for 100+ patients/day and reduced data-entry errors  
- Mentored junior interns on OPD workflows and eligibility verification processes
